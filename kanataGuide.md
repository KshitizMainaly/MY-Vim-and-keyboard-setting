# Kanata Power-User Guide

Your `kanata.kbd` turns Caps Lock into a powerful modifier key. All keys work normally when Caps is not held.

---

## Caps + Navigation (Vim Arrows)
| Combo | Action |
|---|---|
| **Caps + H** | ← Left |
| **Caps + J** | ↓ Down |
| **Caps + K** | ↑ Up |
| **Caps + L** | → Right |

---

## Caps + Editing
| Combo | Action |
|---|---|
| **Caps + D** | Backspace (delete char behind) |
| **Caps + F** | Delete forward (delete char ahead) |
| **Caps + W** | Delete whole word behind (`Ctrl+Bspc`) |

---

## Caps + Common Shortcuts
| Combo | Action |
|---|---|
| **Caps + A** | Select All (`Ctrl+A`) |
| **Caps + S** | Save (`Ctrl+S`) |
| **Caps + C** | Copy (`Ctrl+C`) |
| **Caps + V** | Paste (`Ctrl+V`) |

---

## Special Features
| Combo | Action |
|---|---|
| **Tap Caps** | Escape (for Helix/Vim) |
| **Caps + I** | Caps-Word (auto-uppercase, turns off after space) |
| **Caps + Space (tap)** | `Ctrl+Space` (WezTerm leader) |
| **Caps + Space (hold) + B** | Launch Brave |
| **Caps + Space (hold) + T** | Launch WezTerm |
| **F4** | Emergency Caps Lock toggle |
| **Tap Shift** | One-Shot Shift (capitalizes next letter only) |
